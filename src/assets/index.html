What's the URL I should use for script tags?

<script src="https://webpack.onface.live/?react"></script>

You'll now have window.pkgzip['react'] available to use.

What are some usage examples?

Pkgzip is useful when you want to use packages with dependencies in any environment. Some examples:

  • Using Codepen or JS Bin to demonstrate a bug in a UI component published on npm
  • Hacking on a frontend idea in your favourite IDE without having to set up a webpack project
  • Anything where you can use a script tag!

Should I use this on my production site?

It's not recommended to use Pkgzip for anything customer facing, because the initial build can take a little white, and subsequent requests served from the cache still have some latency.

Are there any restrictions?

There is a request timeout of 60 seconds, which is suitable for typical prototyping. Subsequent requests for the same bundle will usually be served from the cache much more quickly.

All packages are retrieved from the public npm registry.
